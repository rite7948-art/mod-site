# Запуск бота — пошагово

## 1. Создай приложение и бота в Discord

1. Зайди на https://discord.com/developers/applications → **New Application** → дай имя (например `Futurama Moder Bot`).
2. В разделе **Bot** → **Reset Token** → скопируй токен (это `DISCORD_TOKEN`).
3. В разделе **General Information** → скопируй **Application ID** (это `CLIENT_ID`).
4. В разделе **Installation** или **OAuth2 → URL Generator**:
   - Scopes: `bot`, `applications.commands`
   - Bot Permissions: `Send Messages` (для ЛС хватит этого)
   - Скопируй URL внизу, открой его и **пригласи бота на свой сервер**.
5. На сервере Discord включи режим разработчика (Настройки → Расширенные → Режим разработчика). Правый клик по серверу → **Скопировать ID** — это `GUILD_ID`.

## 2. Положи бота на Railway

1. В том же проекте Railway, где уже сидят футика2 и (после деплоя) сайт модераторов — **New → Service → Empty Service**.
2. Подключи к этому Git-репозиторию.
3. **Settings → Build → Dockerfile Path** → укажи `Dockerfile.bot`.
4. **Variables** — добавь:
   ```
   DISCORD_TOKEN=твой_токен_бота
   CLIENT_ID=айди_приложения
   GUILD_ID=айди_сервера
   SITE_URL=https://твой-сайт.up.railway.app
   USERS_JSON_PATH=/data/users.json
   ```
5. **Volumes** — создай Volume и подмонти его на путь `/data`. **Тот же Volume** подмонти к сервису сайта (тоже в `/data`). Так бот пишет, сайт читает — один и тот же файл.
6. Деплой. В логах должно появиться:
   ```
   🤖 Бот запущен как Futurama Moder Bot#1234
   📂 users.json: /data/users.json
   ✅ Команды зарегистрированы на сервере 123...
   ```

## 3. Используй

В Discord на твоём сервере введи:

```
/модерсайт-доступ юзер:@Имя логин:vasya роль:Куратор
```

Бот:
- Сгенерирует пароль
- Запишет в `/data/users.json`
- Отправит юзеру в ЛС: логин/пароль/роль/ссылку на сайт
- Тебе ответит галочкой (приватно)

## Все команды

| Команда | Что делает |
|---|---|
| `/модерсайт-доступ юзер логин роль` | Создать новый аккаунт |
| `/модерсайт-сбросить логин` | Сгенерировать новый пароль (старый перестанет работать) |
| `/модерсайт-удалить логин` | Удалить аккаунт |
| `/модерсайт-список` | Показать всех пользователей сайта (только тебе) |

Все команды доступны только пользователям с правами **Администратор** на сервере.

## Локальная разработка бота (опционально)

```bash
# в папке проекта
npm install

# создай .env с переменными (см. .env.example)

node bot.js
```

Без Railway Volume `users.json` будет создаваться рядом с `bot.js`.
